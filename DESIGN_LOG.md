# RoboArm UI — Лог дизайна интерфейса

> Дата: 19 марта 2026  
> Проект: digital-twin  
> Задача: Спроектировать и сверстать desktop + mobile интерфейс управления роборукой

---

## Концепция

**Направление:** Industrial Command Center — промышленный командный пульт нового поколения.

Не типичный "кибер-неон", а зрелая утилитарная эстетика: тёплые металлические тона, чёткая структура, ощущение настоящего инженерного инструмента. Как пульт управления ЧПУ, но с современной типографикой и анимациями.

**Ключевой визуальный приём:** янтарный акцент (#F59E0B) на глубоком тёмном фоне — ассоциация с промышленными индикаторами и приборными панелями.

## Компоненты интерфейса

1. **Header** — sticky, с логотипом, названием и статус-пилом подключения
2. **Connection Panel** — выбор baud rate, кнопки подключения/отключения
3. **Packet Display** — live-превью формируемого пакета данных
4. **Arm Visualization** — SVG-схема манипулятора с подсвечиваемыми суставами
5. **Servo Sliders ×5** — слайдеры с track fill, числовым значением, ID-бейджем
6. **Serial Monitor** — лог с color-coded записями (TX/RX/SYS/ERR)
7. **Manual Input** — поле ввода произвольных команд
8. **Auto-send Toggle** — переключатель авто-отправки
9. **Mobile Bottom Nav** — 4-табовая навигация для мобильной версии
10. **Mobile Menu** — hamburger-меню для доступа к sidebar

---

## Интерактивные элементы (только UI, без управления)

- Слайдеры обновляют значения + track fill в реальном времени
- Подсветка суставов на SVG при изменении слайдера (glow, 400ms)
- Packet preview мгновенно перестраивается
- Demo-режим подключения (UI state toggle)
- Demo-записи в Serial Monitor
- Переключение мобильных табов
- Toggle авто-отправки

---

## Что НЕ реализовано (по условию)

- Web Serial API подключение
- Реальная отправка данных на Arduino
- Парсинг входящих данных
- Сохранение/восстановление позиций
- Анимация SVG-схемы в зависимости от углов (forward kinematics)
