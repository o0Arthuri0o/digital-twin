7. Интерактивное поведение

  • Слайдеры обновляют значения в реальном времени + track fill
  • При изменении слайдера подсвечивается соответствующий сустав на SVG-схеме (glow-эффект, 400ms)
  • Packet preview обновляется мгновенно при движении слайдера
  • Toggle «авто-отправка» — при включении данные отправляются на каждый input event
  • Serial Monitor — новые записи появляются с анимацией fade-in-up (200ms)
  • Кнопки: scale(0.97) при active, translateY(-1px) при hover для primary
  • Mobile: переключение табов через bottom navigation
  • Mobile: sidebar и monitor открываются как full-screen overlays

8. Брейкпоинты

  • ≥1100px — 3-колоночный layout: sidebar (260px) + content (flex) + monitor (300px)
  • 768–1100px — 2 колонки: sidebar + content, monitor снизу
  • ≤768px — мобильный: одноколоночный, bottom nav, overlay panels
  • ≤420px — компактный мобильный: уменьшенные элементы
